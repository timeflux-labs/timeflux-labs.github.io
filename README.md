# StorySeek - AI增强的中长篇小说创作

StorySeek 是一款专为网络小说作者打造的 AI 辅助创作平台，提供从设定集、大纲、细纲到正文的一站式写作体验。

---

## 产品特色

- 遵从创作习惯：从设定到正文生成（人设 / 世界观 / 纲要剧情）
- 全流程辅助：设定 → 大纲 → 章纲 → 正文，可自由调整创作流程
- AI 无痕创作：更像人写的内容
- 提示词生态：创作秘籍、经验模板支持共享分润，持续激励社区贡献。
- 极致性价比：以人为本，致力于最小token消耗

---

## 产品愿景

我们希望让 AI 成为创作者的得力助手，而非替代者。降低小白写作门槛，也为专业作者节省脑力负担。

## 技术栈

- 前端：Vue 3 + Tailwind CSS
- 后端：Spring Boot + Spring AI + MyBatis 
- AI 接入：OpenAI / Gemini
